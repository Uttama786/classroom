======================================================================
  Web Development – Quick Notes
  FlipLearn | Quick Reference Notes | 2025-26
======================================================================


────────────────────────────────────────────────────────────
  HTML5 Essentials
────────────────────────────────────────────────────────────
  Document structure: <!DOCTYPE>, <html>, <head>, <body>.
  Semantic elements: header, nav, main, article, section, footer.
  Forms, input types (text, email, date, file), validation attributes.

────────────────────────────────────────────────────────────
  CSS3 & Responsive Design
────────────────────────────────────────────────────────────
  Selectors, specificity, cascade, inheritance, box model.
  Display: block/inline/flex/grid. Position:
  static/relative/absolute/fixed.
  Flexbox and CSS Grid. Media queries, Bootstrap 5 grid and utilities.

────────────────────────────────────────────────────────────
  JavaScript & DOM
────────────────────────────────────────────────────────────
  var/let/const, functions, arrow functions, closures, promises,
  async/await.
  DOM: querySelector, getElementById, createElement, event listeners.
  fetch() API, JSON handling, event delegation, localStorage.

────────────────────────────────────────────────────────────
  Django Framework
────────────────────────────────────────────────────────────
  MVT pattern: Model – View – Template.
  URL routing (urls.py), class-based vs function-based views.
  ORM: models, migrations, QuerySets, Q objects, aggregation.

────────────────────────────────────────────────────────────
  REST APIs & AJAX
────────────────────────────────────────────────────────────
  HTTP methods: GET, POST, PUT, PATCH, DELETE. Status codes.
  Django REST Framework: serializers, viewsets, routers, permissions.
  Consuming APIs from the front end with fetch / Axios.

────────────────────────────────────────────────────────────
  Deployment
────────────────────────────────────────────────────────────
  Static files with collectstatic and WhiteNoise.
  gunicorn as WSGI server. Environment variables and .env files.
  Deploying to Render / Railway: Procfile, runtime.txt, build
  commands.
