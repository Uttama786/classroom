======================================================================
  Python Programming – Quick Notes
  FlipLearn | Quick Reference Notes | 2025-26

────────────────────────────────────────────────────────────
  Basics
────────────────────────────────────────────────────────────
  Variables, data types (int, float, str, bool, NoneType), operators.
  Type conversion: int(), float(), str(), bool().
  Input/output: input() and print() with f-strings.

────────────────────────────────────────────────────────────
  Control Flow
────────────────────────────────────────────────────────────
  if-elif-else, while loop, for loop over iterables.
  break, continue, pass. List/dict/set comprehensions.
  Exception handling: try-except-else-finally, custom exceptions.

────────────────────────────────────────────────────────────
  Functions & Modules
────────────────────────────────────────────────────────────
  def, return, *args, **kwargs, default and keyword arguments.
  Lambda functions, map(), filter(), functools.reduce().
  Modules, packages, __name__ == '__main__', import system.

────────────────────────────────────────────────────────────
  Object-Oriented Programming
────────────────────────────────────────────────────────────
  Classes, objects, __init__, self, class vs instance attributes.
  Inheritance, multiple inheritance, MRO, polymorphism, encapsulation.
  Magic/dunder methods: __str__, __repr__, __len__, __eq__, __iter__.

────────────────────────────────────────────────────────────
  File Handling & Standard Library
────────────────────────────────────────────────────────────
  open(), read(), readline(), write(), with statement (context
  manager).
  os, sys, pathlib, datetime, math, random, json, csv modules.
  pip, virtual environments (venv), requirements.txt, pyproject.toml.

────────────────────────────────────────────────────────────
  Advanced Python
────────────────────────────────────────────────────────────
  Decorators (@wraps), generators (yield), iterators
  (__iter__/__next__).
  Context managers (__enter__/__exit__), metaclasses, descriptors.
  Async/await basics: asyncio, coroutines, event loop, gather().
