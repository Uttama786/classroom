────────────────────────────────────────────────────────────
  Introduction to AI
────────────────────────────────────────────────────────────
  History of AI, Turing Test, AI winters, modern renaissance.
  AI vs ML vs DL. Narrow vs General vs Superintelligence.
  Uninformed search: BFS, DFS. Informed search: greedy, A*.

────────────────────────────────────────────────────────────
  Supervised Learning
────────────────────────────────────────────────────────────
  Regression: linear, polynomial, ridge (L2), lasso (L1), elastic net.
  Classification: k-NN, Naive Bayes, SVM (kernel trick), decision
  trees.
  Ensemble: bagging (Random Forest), boosting (AdaBoost, XGBoost,
  LightGBM).

────────────────────────────────────────────────────────────
  Unsupervised Learning
────────────────────────────────────────────────────────────
  Clustering: K-Means (elbow method), DBSCAN (epsilon/minPts),
  hierarchical (dendrograms).
  Association rules: Apriori, FP-Growth, support, confidence, lift.
  Dimensionality reduction: PCA, t-SNE, autoencoders.

────────────────────────────────────────────────────────────
  Neural Networks
────────────────────────────────────────────────────────────
  Perceptron, multi-layer perceptron (MLP), activation functions
  (ReLU, sigmoid, tanh, softmax).
  Backpropagation algorithm, gradient descent variants (SGD, Momentum,
  Adam, RMSProp).
  Overfitting remedies: dropout, batch normalisation, L1/L2
  regularisation, early stopping.

────────────────────────────────────────────────────────────
  Deep Learning
────────────────────────────────────────────────────────────
  CNNs: convolution, pooling, stride, padding. Architectures: VGG,
  ResNet, InceptionNet.
  RNNs: vanishing gradient problem. LSTM and GRU gating mechanisms.
  Transformers: self-attention, multi-head attention, positional
  encoding, BERT, GPT.

────────────────────────────────────────────────────────────
  Model Evaluation & Deployment
────────────────────────────────────────────────────────────
  Bias-variance tradeoff, learning curves, validation curves.
  Hyperparameter tuning: grid search, random search, Bayesian
  optimisation (Optuna).
  Model serving: REST API (FastAPI/Flask), ONNX export, TensorFlow
  Serving, MLflow.
