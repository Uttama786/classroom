────────────────────────────────────────────────────────────
  Network Models
────────────────────────────────────────────────────────────
  OSI 7-layer model: Physical, Data Link, Network, Transport, Session,
  Presentation, Application.
  TCP/IP 4-layer model and mapping to OSI layers.
  Encapsulation (sender) and decapsulation (receiver).

────────────────────────────────────────────────────────────
  Data Link Layer
────────────────────────────────────────────────────────────
  Framing, error detection (CRC, checksum, parity), error correction
  (Hamming).
  MAC addresses, ARP (Address Resolution Protocol), Ethernet (IEEE
  802.3).
  Switches, VLANs, Spanning Tree Protocol (STP), PoE.

────────────────────────────────────────────────────────────
  Network Layer & IP
────────────────────────────────────────────────────────────
  IPv4: 32-bit addresses, subnetting, CIDR notation, private ranges.
  IPv6: 128-bit addresses, EUI-64, stateless autoconfiguration.
  Routing: static routes, RIP, OSPF, BGP. NAT, PAT, ICMP.

────────────────────────────────────────────────────────────
  Transport Layer
────────────────────────────────────────────────────────────
  TCP: 3-way handshake (SYN/SYN-ACK/ACK), 4-way teardown, flow control
  (sliding window).
  Congestion control: slow start, congestion avoidance, fast
  retransmit.
  UDP: connectionless, best-effort, low overhead. Ports and sockets.

────────────────────────────────────────────────────────────
  Application Layer
────────────────────────────────────────────────────────────
  DNS: hierarchy, recursive vs iterative queries, A/AAAA/MX/CNAME
  records.
  HTTP/1.1 vs HTTP/2 vs HTTP/3. FTP, SMTP, POP3, IMAP, DHCP, SSH.
  TLS/SSL: handshake, certificates, cipher suites.

────────────────────────────────────────────────────────────
  Network Security
────────────────────────────────────────────────────────────
  Threats: DoS/DDoS, MitM, ARP spoofing, phishing, SQL injection.
  Defences: firewalls (stateful/stateless), IDS/IPS, VPN, IPSec.
  Cryptography: symmetric (AES), asymmetric (RSA), PKI, digital
  signatures.
